Frequently Asked Questions

Is ScavHub free?

ScavHub is a free product, but it also offers a premium version with additional features!

How much does premium cost?

Premium costs 115 Robux. If you prefer PayPal, it’s $2, or you can pay with $2 worth of Ethereum.

I bought the script. How do I use it?

After purchasing the script, you need to get whitelisted. Please create a ticket in our Discord server, provide proof of purchase, and include your Roblox user ID.

Can I whitelist using HWID?

Yes! Simply request HWID whitelisting, and we will assist you through the process.

What’s the difference between HWID and ID whitelisting?

HWID whitelisting allows you to use multiple alts without needing to whitelist each one individually, whereas ID whitelisting is limited to specific accounts.

How many accounts can I whitelist using ID whitelisting?

You can whitelist up to 2 accounts using ID whitelisting. If you need to whitelist another ID, you will need to remove one of the existing IDs (of your choice).
